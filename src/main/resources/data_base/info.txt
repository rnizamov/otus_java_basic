Разработана структура БД для проведения тестирования пользователей.

Таблицы и описание полей:

1 test
test_name - имя теста
description - описание
level - сложность
group_test_id - к какой группе принадлежит тест

2 group_test
group_name имя группы тестов

3 question
question - вопрос
test_id - к какому тесту относится запись

4 answer
answer - ответ
istrue - правда или ложь

5 user
first/last/middle_name - имя фамилия отчество

6 user_rating
user_id - юзер
test_id - тест
rating - оценка за тест
answered_questions - вопросы на которые ответил пользователь правильно(массив integer)
